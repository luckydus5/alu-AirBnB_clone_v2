# web_static

Static HTML/CSS pages for the AirBnB clone (the web-static project).

These files are packed and deployed to the web servers by the Fabric
scripts at the repository root (`1-pack_web_static.py`,
`2-do_deploy_web_static.py`, `3-deploy_web_static.py`).

- `*.html` — the static pages.
- `styles/` — CSS files.
- `images/` — image assets.
