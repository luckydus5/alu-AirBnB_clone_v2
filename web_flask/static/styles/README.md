# web_flask/static/styles

CSS for the HBNB filters page.

- `4-common.css` — base layout and typography.
- `3-header.css`, `3-footer.css` — page header and footer.
- `6-filters.css` — the filters bar and popovers. The `.popover` class is
  capped at `max-height: 300px` with `overflow: auto` for scrolling.
