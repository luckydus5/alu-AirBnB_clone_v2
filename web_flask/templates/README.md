# web_flask/templates

Jinja templates rendered by the `web_flask` routes.

- `5-number.html`, `6-number_odd_or_even.html` — number display pages.
- `7-states_list.html`, `8-cities_by_states.html`, `9-states.html` —
  pages listing states and their cities from storage.
- `10-hbnb_filters.html` — the HBNB filters page with State, City and
  Amenity data.
